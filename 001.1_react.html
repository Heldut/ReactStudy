No React, dentro do elemento "createElement", podemos criar vários outros
elementos, isto é, aninhar outros elementos.
Por exemplo, no primeiro elemento:

         React.createElement(
                  "h1",
                  null,
                  "Olá Mundo! By React"
         ),

Poderiamos colocar o texto "By React" em outro elemento:

         React.createElement(
                  "h1",
                  null,
                  "Olá Mundo!"
                  React.createElement("small", null, " By React!")
         ),

Podemos colocar quantos elementos quisermos...
Como vemos no exemplo acima, podemos colocar uma quebra de linha:

         React.createElement(
                  "h1",
                  null,
                  "Olá Mundo!"
                  React.createElement("br", null, " By React!")
                  React.createElement("small", null, " By React!")
         ),

Para finalizar essa aula é mostrado como colocar o atributo no elemento "createElement".
Para isso criamos nosso "stylo" css e informamos a classe usando o atributo "className" no
segundo parâmetro do elemento "createElement". No exemplo criei uma classe "smallColor" com a
cor vermelha e adicionei no segundo parâmetro do elemento "small":

                  React.createElement("small", { className: "smallColor" }, " By React!")


Um ponto que vale colocar aqui é o uso de um namespace. Podemos criar uma variável
que receberá o "createElement" e não precisaremos ficar a todo tempo escrevendo
"React.createElement".

Com isso finalizo a primeira aula. Abaixo como está nosso código:

<html>
    <head>
        <title>001 - React Olá Mundo!</title>
         <style>
             .smallColor{
                  color: red;
             }
         </style>
    <head/>
         <body>

             <div id="root"></div>

             <script crossorigin src="https://unpkg.com/react@17/umd/react.development.js"></script>
             <script crossorigin src="https://unpkg.com/react-dom@17/umd/react-dom.development.js"></script>
             <script>
                  var e = React.createElement;
                  ReactDOM.render(
                       e(
                           "h1",
                           null,
                           "Olá Mundo!",
                           e("br"),
                           e("small", { className: "smallColor" }, " By React!")
                       ),
                       document.getElementById("root")
                  );
             </script>
         </body>
     </html>
