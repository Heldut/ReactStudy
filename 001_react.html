Para iniciarmos, dentro do das tags script, vamos inserir a biblioteca ReactDOM,
que fará a conexão do React com o DOM. Vamos usar o método render:

ReactDOM.render();

O método "render" recebe 2 parâmetros. O primeiro é o que vamos renderizar.
O segundo vamos informar onde queremos renderizar. Bem simples, o que e onde vamos renderizar
as informações.
Para informar o que vamos renderizar vamsos utilizar a biblioteca "createElement" do React.

ReactDOM.createElement();

O método "createElement" recebe 3 parâmetros:
No primeiro, qual a tag que vamos renderizar. O segundo é o atributo, caso exista e no terceiro
o filho do elemento, o texto.

Agora vamos informar "onde" queremos renderizar. Nesse caso vamos criar uma
tag <div> com o respectivo ID e informar no método getElementById
do Javascript:

document.getElementById("root)"

Assim, nosso html deve ter a seguinte aparência:

<html>
    <head>
        <title>001 - React Olá Mundo!</title>
    <head/>
    <body>

        <div id="root">& lt;/div>

        <script crossorigin src="https://unpkg.com/react@17/umd/react.development.js"></script>
        <script crossorigin src="https://unpkg.com/react-dom@17/umd/react-dom.development.js"></script>
        <script>

                 ReactDOM.render(

                           React.createElement(
                                    "h1",
                                    null,
                                    "Olá Mundo! By React"
                           ),
                           document.getElementById("root")
                 );

        </script>
    </body>
</html>
