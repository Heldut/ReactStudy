Como estamos utilizando o Babel podemos mudar algumas nomeclaturas atuais como o arrow function.
 Para isso vamos fazer as alterações necessárias...
 Anteriormente usavamos a palavra "var" para criar variáveis. Agora utilizamos "let" quando quisermos alterar a variável
 e "const" quando não formos alterar.
 Assim, vamos utilizar a "arrow function". Nossa função sera alterada como mostrado abaixo:

   function Header(){
       return(
           <div className="header">Calculadora</div>
       );
   }

   const Header = () => {
       return(
           <div className="header">Calculadora</div>
       );
   }


Quando vamos retornar apenas uma linha não há a necessidade de escrever o return...
